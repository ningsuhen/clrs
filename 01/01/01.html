Exercise 1.1.1

Give a real-world example that requires sorting or a real-world example that requires computing a convex hull.

A real-world example of sorting is pretty straightforward - for example, every web page that renders a list alphabetically needs to sort the entries - a catalog, an index or whatever.

I have no clue why we need to compute convex hulls, but Wikipedia says that there are a bunch of applications.
